# 아마존 판매 데이터 분석 (2022-2023)
분석 기간 : 2026-02-11 ~ 2026-02-14  
## Power BI 대시보드

![카테고리별 분석 대시보드](스크린샷%202026-02-13%20220234.png)

> 카테고리 선택 필터를 통해 연도별 매출, 판매량, 결제 방식, 지역 분포 및
> 월별 비교를 확인할 수 있도록 구성

## 프로젝트 개요
Amazon 판매 데이터를 활용하여  
카테고리별 매출 성장 구조와 주요 영향 요인을 분석하였습니다.

단순 매출 비교가 아닌,  
**주문 건수(수요)와 평균 주문 금액(AOV)의 구조적 분해**를 통해  
성장의 실제 원인을 도출하는 것을 목표로 하였습니다.

## 활용 데이터
Amazon Sales Dataset : https://www.kaggle.com/datasets/aliiihussain/amazon-sales-dataset

## 데이터 구조
-   order_date : 주문 일자  
-   product_category : 제품 카테고리  
-   quantity_sold : 구매 수량  
-   customer_region : 고객 지역  
-   payment_method : 결제 방식  
-   rating : 상품 평점  
-   review_count : 리뷰 수  
-   discounted_price : 할인 적용 가격  
-   total_revenue : 총 결제 금액

## 데이터 확인
- 50000개의 데이터
- 데이터 수집 기간 : 2022-01-01 ~ 2023-12-31
- 6개의 카테고리로 구성  
    Beauty(뷰티), Books(도서), Electronics(전자), 
    Fashion(의류), Home&Kitchen(생활·주방용품), Sports(스포츠)  

## 데이터 분석
### 1. 매출량에 할인이 영향이 있는가?
|카테고리       |  주문 성장률  | 매출 성장률 |  할인율 변화|
|----------------| -------------| -------------| -------------|
|Beauty           |+2.75%        |+2.83%        |-0.01%|
|Books            |+0.22%        |+1.70%        |+0.02%|
|Electronics      |+1.40%        |+0.81%        |-0.32%|
|Fashion          |+1.42%        |-1.05%        |+0.23%|
|Home & Kitchen   |+0.63%        |+0.31%        |-0.07%|
|Sports           |-2.84%        |-1.38%        |-0.06%|

#### 스피어만 상관계수
##### 판매량 성장에 할인의 영향  
Spearman = 0.2 -> 약한 상관 관계 // p-value(0.704) > 0.05 -> 유의X  
판매량에 할인은 영향을 끼치지 않는다. 
##### 매출 성장에 할인의 영향  
Spearman = 0.028 -> 관계 X // p-value(0.957) > 0.05 -> 유의X  
매출에 할인은 영향을 끼치지 않는다.  

### 2. 카테고리별 매출 성장 구조 분석

|카테고리         |주문 성장률   |매출 성장률   |객단가 차이|
|---------------- |------------- |------------- |---------------------|
|Beauty           |+2.75%        |+2.83%        |+0.07%|
|Books            |+0.22%        |+1.70%        |+1.48%|
|Electronics      |+1.40%        |+0.81%        |-0.59%|
|Fashion          |+1.42%        |-1.05%        |-2.43%|
|Home & Kitchen   |+0.63%        |+0.31%        |-0.32%|
|Sports           |-2.84%        |-1.38%        |+1.50%|

#### 성장률 분석 결과
Beauty : 수요 증가로 인한 매출 상승  
Books : 주문량 변화는 미미하나 평균 주문 금액 상승으로 인한 매출 증가  
Electronics : 주문 증가 중심의 성장이나 객단가 하락 일부 상쇄  
Fashion : 주문은 증가했으나 평균 주문 금액 하락으로 매출 감소  
Home & Kitchen : 주문 증가와 객단가 하락이 상쇄된 안정적 구조  
Sports : 평균 주문 금액은 상승했으나 수요 감소로 매출 하락

## 핵심 결론
-   할인율 변화는 매출 성장과 유의한 관계가 없음  
-   매출 변화는 주로 **주문 수(수요) 변화와 평균 주문 금액(AOV)의 구조적
    변동**에 의해 결정됨  
-   카테고리별로 서로 다른 성장 유형이 존재

## 한계점
-   product_id가 고유 식별자로 사용되기 어려운 데이터 구조  
-   외부 요인(경기, 시즌성 등)은 반영되지 않음

## 활용 기술
-   Python : 데이터 분석 및 전처리  
-   pandas, numpy : 데이터 처리  
-   MySQL : 데이터 구조 및 품질 확인  
-   Power BI : 대시보드 제작
